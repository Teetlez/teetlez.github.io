Path-tracing


by Timothy Clocksin

Discription

A monte carlo path-tracer written in Rust following the "Raytracing in One Weekend" book. I used this project both as a way to build my skills with rendering and to better learn Rust. at the moment it is not usable inside the website. It is still a work in progress and I want to finish a few features before releasing it.
