3D Wireframe


by Timothy Clocksin

Discription

This code uses no 3D rendering libraries or functions. It was purely built from scratch in order to demonstrate knowledge of the process used to render wireframes in 3D. However, shading was not implemented due to limitations in the flexability of the processing.js web enviorment.

For more information, go to:

www.khanacademy.org/computing

Controls

  • UP & DOWN - Rotate vertically
  • LEFT & RIGHT - Rotate horizontally
  • Ctrl - Rotate Counter-clockwise
  • Alt - Rotate Clockwise
